# Data Visualization with Python

This repository contains hands-on projects and notebooks focused on data storytelling and visual analysis using Python's most powerful plotting libraries.

## Libraries Used

* **Matplotlib**: The foundation for static, animated, and interactive visualizations in Python.
* **Seaborn**: High-level interface for drawing attractive and informative statistical graphics.

## Visualizations Included

This project covers various plot types to identify patterns and trends:
* **Distribution Plots**: Histograms and KDE plots to understand data spread.
* **Relational Plots**: Scatter plots and line graphs for correlation analysis.
* **Categorical Plots**: Bar charts, box plots, and violin plots for comparing groups.
* **Matrix Plots**: Heatmaps to visualize correlations between variables.

## Environment
* **Platform**: Google Colab
* **Language**: Python
